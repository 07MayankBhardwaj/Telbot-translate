# Assets Directory

This directory contains application icons and images.

## Required Files

For production, you'll need:
- `icon.png` - Main app icon (256x256 or larger)
- `icon.ico` - Windows icon
- `tray-icon.png` - System tray icon (16x16 or 32x32)

## Generating Icons

You can use tools like:
- https://converticon.com/ - PNG to ICO converter
- https://www.icoconverter.com/ - Multi-size ICO generator
